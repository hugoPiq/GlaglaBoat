#### Pré-requis
VREP
ROS

#### Installation

- Cloner le dépot.

#### Démarrage
- Lancer ROSCORE en premier:
_exemple_: Executez la commande ``roscore`` .
- Lancer VREP puis ouvrir la scène ``real_boat.ttt``.
- Lancer RQT.
_exemple_: Executez la commande ``rqt`` .
